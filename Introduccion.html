Introducción a LINQ

¿Qué es LINQ y por qué hace tu código más limpio y expresivo?

¿Qué es LINQ?

LINQ (Language Integrated Query) es una característica de C# que permite realizar consultas directamente sobre colecciones de datos usando una sintaxis integrada en el lenguaje.

Con LINQ puedes trabajar sobre listas, arreglos, bases de datos, archivos XML y más, todo con la misma sintaxis.

💡 En resumen: LINQ es como tener SQL directamente dentro de C#, pero aplicable a cualquier tipo de colección.

Namespace necesario

Para usar LINQ debes agregar este using al inicio de tu archivo:

C# — using requerido
using System;
using System.Collections.Generic;
using System.Linq;  // ← Este es el que activa LINQ

Dos sintaxis equivalentes

LINQ tiene dos formas de escribirse — ambas producen exactamente el mismo resultado:

1. Sintaxis de consulta (estilo SQL)

C# — Query Syntax
int[] numeros = { 5, 3, 8, 1, 9, 2, 7 };

var resultado = from n in numeros
                where n > 4
                orderby n
                select n;

// Resultado: 5 7 8 9

2. Sintaxis de métodos (recomendada)

C# — Method Syntax
int[] numeros = { 5, 3, 8, 1, 9, 2, 7 };

var resultado = numeros
    .Where(n => n > 4)
    .OrderBy(n => n);

// Resultado: 5 7 8 9
✅ Recomendación: Se prefiere la sintaxis de métodos en el mundo profesional porque es más flexible y compatible con todos los operadores LINQ.

Expresiones Lambda

La sintaxis de métodos usa expresiones lambda para definir condiciones. Son funciones anónimas compactas:

C# — Lambda básica
// Forma:  parametro => expresión
n => n > 4          // n es mayor a 4?
p => p.Precio       // la propiedad Precio de p
e => e.Departamento // la propiedad Departamento de e
