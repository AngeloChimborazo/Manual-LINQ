Filtrado básico — Temperaturas

Enunciado: Dada una lista de temperaturas en °C { 35, 18, 42, 7, 28, 15, 50, 22 }, obtén las temperaturas "calurosas" (mayores a 30°C) y muéstralas en orden descendente.

Operadores a usar: Where, OrderByDescending

C# — Solución
List<int> temperaturas = new List<int> { 35, 18, 42, 7, 28, 15, 50, 22 };

var calurosas = temperaturas
    .Where(t => t > 30)
    .OrderByDescending(t => t);

Console.WriteLine("Temperaturas calurosas (desc):");
foreach (var t in calurosas)
    Console.WriteLine($"  {t}°C");

// Salida:
// 50°C
// 42°C
// 35°C

Agregación — Ventas diarias

Enunciado: Tienes las ventas diarias de una semana: { 1500, 2300, 800, 4100, 950, 3200, 1750 }. Calcula: total, venta máxima, mínima y promedio.

Operadores a usar: Sum, Max, Min, Average

C# — Solución
List<double> ventas = new List<double>
    { 1500, 2300, 800, 4100, 950, 3200, 1750 };

Console.WriteLine($"Total de ventas:  ${ventas.Sum():F2}");
Console.WriteLine($"Venta más alta:   ${ventas.Max():F2}");
Console.WriteLine($"Venta más baja:   ${ventas.Min():F2}");
Console.WriteLine($"Venta promedio:   ${ventas.Average():F2}");

// Salida:
// Total de ventas:  $16600.00
// Venta más alta:   $4100.00
// Venta más baja:   $800.00
// Venta promedio:   $2371.43

LINQ con objetos — Biblioteca de libros

Enunciado: Crea una lista de 5 libros con Titulo, Autor, Anio y Precio. Luego: muestra los libros publicados después de 2015, el libro más barato, y todos ordenados por precio.

Operadores a usar: Where, OrderBy, First

C# — Solución
class Libro
{
    public string Titulo { get; set; }
    public string Autor  { get; set; }
    public int    Anio   { get; set; }
    public double Precio { get; set; }
}

List<Libro> libros = new List<Libro>
{
    new Libro { Titulo="Clean Code",              Autor="Robert C. Martin", Anio=2008, Precio=45.00 },
    new Libro { Titulo="The Pragmatic Programmer",Autor="Hunt and Thomas",  Anio=2019, Precio=52.00 },
    new Libro { Titulo="C# in Depth",            Autor="Jon Skeet",        Anio=2019, Precio=48.00 },
    new Libro { Titulo="Design Patterns",         Autor="GoF",              Anio=1994, Precio=39.00 },
    new Libro { Titulo="You Don't Know JS",       Autor="Kyle Simpson",     Anio=2018, Precio=30.00 }
};

// Libros desde 2015
Console.WriteLine("=== Libros desde 2015 ===");
foreach (var l in libros.Where(l => l.Anio > 2015).OrderBy(l => l.Anio))
    Console.WriteLine($"  [{l.Anio}] {l.Titulo} - ${l.Precio}");

// Libro mas barato
var masBarato = libros.OrderBy(l => l.Precio).First();
Console.WriteLine($"\n=== Mas barato: {masBarato.Titulo} (${masBarato.Precio}) ===");

// Todos ordenados por precio
Console.WriteLine("\n=== Ordenados por precio ===");
foreach (var l in libros.OrderBy(l => l.Precio))
    Console.WriteLine($"  ${l.Precio:F2} - {l.Titulo}");

GroupBy + Agregación — Inventario de tienda

Enunciado: Tienes un inventario de productos con categoría y precio. Agrupa por categoría y muestra cuántos productos hay en cada una y el precio promedio.

Operadores a usar: GroupBy, Select, Count, Average

C# — Solución
class ProductoTienda
{
    public string Categoria { get; set; }
    public string Nombre    { get; set; }
    public double Precio    { get; set; }
}

List<ProductoTienda> inventario = new List<ProductoTienda>
{
    new ProductoTienda { Categoria="Electronico", Nombre="Laptop",   Precio=1200 },
    new ProductoTienda { Categoria="Electronico", Nombre="Tablet",   Precio=350  },
    new ProductoTienda { Categoria="Ropa",        Nombre="Camiseta", Precio=25   },
    new ProductoTienda { Categoria="Ropa",        Nombre="Pantalon", Precio=55   },
    new ProductoTienda { Categoria="Ropa",        Nombre="Chaqueta", Precio=120  },
    new ProductoTienda { Categoria="Electronico", Nombre="Mouse",    Precio=30   },
    new ProductoTienda { Categoria="Alimento",    Nombre="Arroz",    Precio=2.5  },
    new ProductoTienda { Categoria="Alimento",    Nombre="Leche",    Precio=1.8  }
};

var resumen = inventario
    .GroupBy(p => p.Categoria)
    .Select(g => new
    {
        Categoria   = g.Key,
        Cantidad    = g.Count(),
        PrecioMedio = g.Average(p => p.Precio)
    })
    .OrderBy(r => r.Categoria);

foreach (var c in resumen)
    Console.WriteLine($"{c.Categoria}: {c.Cantidad} items, promedio ${c.PrecioMedio:F2}");

// Salida:
// Alimento:    2 items, promedio $2.15
// Electronico: 3 items, promedio $526.67
// Ropa:        3 items, promedio $66.67

Caso real — Sistema de Infracciones de Tráfico

Contexto: Tienes una lista de infracciones de tráfico con los campos: Ciudad, MarcaVehiculo, AñoVehiculo, TipoVehiculo, FechaInfraccion, TipoInfraccion, MontoMulta e InteresMulta. Resuelve las siguientes consultas usando LINQ.

C# — Clase entidad
class Infraccion
{
    public string   Ciudad          { get; set; }
    public string   MarcaVehiculo   { get; set; }
    public int      AnoVehiculo     { get; set; }
    public string   TipoVehiculo    { get; set; }
    public DateTime FechaInfraccion { get; set; }
    public string   TipoInfraccion  { get; set; }
    public decimal  MontoMulta      { get; set; }
    public decimal  InteresMulta    { get; set; }
}

Consulta A — Top 30 por interés de multa (descendente)

C# — Solución A
var top30 = infracciones
    .OrderByDescending(i => i.InteresMulta)
    .Take(30);

foreach (var i in top30)
    Console.WriteLine($"{i.Ciudad} | {i.MarcaVehiculo} | ${i.InteresMulta}");

Consulta B — Promedio de interés de multa por tipo de infracción

C# — Solución B
var promedioPorTipo = infracciones
    .GroupBy(i => i.TipoInfraccion)
    .Select(g => new
    {
        TipoInfraccion = g.Key,
        Promedio       = g.Average(i => i.InteresMulta)
    });

foreach (var item in promedioPorTipo)
    Console.WriteLine($"{item.TipoInfraccion}: ${item.Promedio:F2}");

Consulta C — Suma de monto por marca (años 2022-2024, meses enero-marzo)

C# — Solución C
var sumaPorMarca = infracciones
    .Where(i => i.AnoVehiculo >= 2022 && i.AnoVehiculo <= 2024
             && i.FechaInfraccion.Month >= 1
             && i.FechaInfraccion.Month <= 3)
    .GroupBy(i => i.MarcaVehiculo)
    .Select(g => new
    {
        Marca = g.Key,
        Suma  = g.Sum(i => i.MontoMulta)
    });

foreach (var item in sumaPorMarca)
    Console.WriteLine($"{item.Marca}: ${item.Suma:F2}");

Consulta D — Promedio de interés de multa por ciudad

C# — Solución D
var promedioPorCiudad = infracciones
    .GroupBy(i => i.Ciudad)
    .Select(g => new
    {
        Ciudad   = g.Key,
        Promedio = g.Average(i => i.InteresMulta)
    });

foreach (var item in promedioPorCiudad)
    Console.WriteLine($"{item.Ciudad}: ${item.Promedio:F2}");

Consulta E — Promedio total a pagar por tipo de vehículo

Filtro: tipo de infracción "suspensión_licencia" y tipo de vehículo "estado".

C# — Solución E
var totalPagar = infracciones
    .Where(i => i.TipoInfraccion == "suspension_licencia"
             && i.TipoVehiculo   == "estado")
    .GroupBy(i => i.TipoVehiculo)
    .Select(g => new
    {
        TipoVehiculo = g.Key,
        TotalPagar   = g.Average(i => i.MontoMulta + i.InteresMulta)
    });

foreach (var item in totalPagar)
    Console.WriteLine($"{item.TipoVehiculo}: ${item.TotalPagar:F2}");
