Capítulo 4 · Ordenamiento

Operadores OrderBy / OrderByDescending

Ordena colecciones de forma ascendente o descendente, por uno o varios criterios.

Resumen de operadores

OperadorDescripción
OrderBy(x => x.Prop)Ascendente (A→Z, 0→9)
OrderByDescending(x => x.Prop)Descendente (Z→A, 9→0)
ThenBy(x => x.Prop)Segundo criterio ascendente
ThenByDescending(x => x.Prop)Segundo criterio descendente

Ejemplo 1 — Orden ascendente

C# — Números de menor a mayor
List<int> numeros = new List<int> { 42, 7, 19, 3, 88, 25 };

var ordenados = numeros.OrderBy(n => n);

foreach (var n in ordenados)
    Console.Write(n + " ");

// Salida: 3 7 19 25 42 88

Ejemplo 2 — Orden descendente

C# — Números de mayor a menor
var descendente = numeros.OrderByDescending(n => n);

foreach (var n in descendente)
    Console.Write(n + " ");

// Salida: 88 42 25 19 7 3

Ejemplo 3 — Ordenar objetos por propiedad

C# — Productos por precio (menor a mayor)
class Producto
{
    public string Nombre { get; set; }
    public double Precio { get; set; }
}

List<Producto> productos = new List<Producto>
{
    new Producto { Nombre = "Laptop",  Precio = 1200 },
    new Producto { Nombre = "Mouse",   Precio = 25   },
    new Producto { Nombre = "Teclado", Precio = 45   }
};

var porPrecio = productos.OrderBy(p => p.Precio);

foreach (var p in porPrecio)
    Console.WriteLine($"{p.Nombre} — ${p.Precio}");

// Salida:
// Mouse — $25
// Teclado — $45
// Laptop — $1200

Ejemplo 4 — Ordenar por texto

C# — Nombres en orden alfabético
List<string> nombres = new List<string>
    { "Carlos", "Ana", "Beatriz", "David", "Alberto" };

var alfabetico = nombres.OrderBy(n => n);

foreach (var n in alfabetico)
    Console.WriteLine(n);

// Salida: Alberto  Ana  Beatriz  Carlos  David

Ejemplo 5 — Múltiples criterios con ThenBy

C# — Por carrera, luego por promedio
class Estudiante
{
    public string Nombre   { get; set; }
    public string Carrera  { get; set; }
    public double Promedio { get; set; }
}

List<Estudiante> estudiantes = new List<Estudiante>
{
    new Estudiante { Nombre="Ana",    Carrera="Software", Promedio=9.2 },
    new Estudiante { Nombre="Carlos", Carrera="Redes",    Promedio=7.8 },
    new Estudiante { Nombre="María",  Carrera="Software", Promedio=8.5 },
    new Estudiante { Nombre="Luis",   Carrera="Redes",    Promedio=9.0 }
};

var ordenados = estudiantes
    .OrderBy(e => e.Carrera)             // 1° por carrera A→Z
    .ThenByDescending(e => e.Promedio);  // 2° por promedio 9→0

foreach (var e in ordenados)
    Console.WriteLine($"{e.Carrera} | {e.Nombre} | {e.Promedio}");

// Salida:
// Redes | Luis   | 9
// Redes | Carlos | 7.8
// Software | Ana   | 9.2
// Software | María | 8.5
